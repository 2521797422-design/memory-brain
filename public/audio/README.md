# Ambient soundscape

On load, the experience plays **Brian Eno — An Ending (Ascent)** as a looping bed:

```
Brian Eno - An Ending (Ascent) [1080 HD].mp3
```

- Starts after your **first click or tap anywhere** on the page (browser audio policy)
- Seamless loop, ~32% volume with a ~5s cinematic fade-in
- Toggle via the soundscape control (bottom-left)

If the file is missing, a built-in procedural drone is used instead.

## Interface sounds

Brain region clicks, navigation buttons, and memory opens play:

```
freesound_community-ui-click-43196.mp3
```

Playback uses soft volume, fade-in/out, and overlap protection. Works independently of the ambient music toggle (top-right).

## Music control

Top-right floating control: ambient music on/off (session memory). Neural interaction sounds stay available once audio is unlocked.
